Cloud Infrastructure Engineer (Entry-Level)

Work Mode: Remote-first with optional hybrid onboarding or team meetups

Focus: Supporting and scaling SurveyMonkey’s AWS-based infrastructure

Responsibilities:

  • Assist in managing cloud environments (AWS, Terraform, Docker)
  • Monitor system performance and automate infrastructure tasks
  • Collaborate with DevOps and security teams to maintain uptime and compliance

Ideal For: CS or IT grads with cloud certifications (e.g., AWS Cloud Practitioner)

Survey Platform QA Analyst

Work Mode: Remote with optional hybrid collaboration during product launches

Focus: Ensuring the quality and functionality of SurveyMonkey’s survey tools

Responsibilities:

  • Test new survey features and integrations (e.g., Microsoft Teams, Slack)
  • Write test cases and report bugs across web and mobile platforms
  • Work closely with product and engineering teams to improve user experience

Ideal For: Grads with a background in software testing, UX, or product QA

SaaS Integration Support Engineer

Work Mode: Remote with optional hybrid for team syncs or client onboarding

Focus: Helping enterprise clients integrate SurveyMonkey with platforms like Microsoft Entra ID, Salesforce, or Oracle

Responsibilities:

  • Guide clients through API usage, SSO setup, and app integrations
  • Troubleshoot technical issues and escalate complex problems
  • Document best practices and contribute to knowledge bases

Ideal For: Entry-level engineers with strong communication skills and a basic understanding of APIs and identity management
